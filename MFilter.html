AngularJS Filters

AngularJS provides filters to transform data:

currency

Format a number to a currency format.

{{ 1234.56 | currency }}

filter

Select a subset of items from an array.

Search:

  • {{ x }}

limitTo

Limits an array/string, into a specified number of elements/characters.

{{ 'AngularJS' | limitTo:5 }}

{{ [1, 2, 3, 4, 5, 6, 7, 8, 9] | limitTo:5 }}

lowercase

Format a string to lower case.

{{ 'AngularJS' | lowercase }}

number

Format a number to a string.

{{ 1234.5678 | number:2 }}

orderBy

Orders an array by an expression.

uppercase

Format a string to upper case.

{{ 'AngularJS' | uppercase }}

Custom Filters

  • {{x | myFormat}}

Make your own filters.
