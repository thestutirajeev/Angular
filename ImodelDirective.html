Use the ng-model directive to bind the value of the input field to a property made in the controller.

Name:

You entered: {{name}}

Change the name inside the input field, and you will see the name in the header changes accordingly.



The ng-model directive can provide type validation for application data (number, e-mail, required):

Enter Something: The field is required.
Email: Not a valid e-mail address Status Valid: {{myForm.myAddress.$valid}} Dirty: {{myForm.myAddress.$dirty}} Touched: {{myForm.myAddress.$touched}}

If the property in the ng-model attribute does not exist, AngularJS will create one for you.
